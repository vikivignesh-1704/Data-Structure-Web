Definition:

A data structure is a storage that is used to store and organize data. It is a way of arranging data on a computer so that it can be accessed and updated efficiently. A data structure is not only used for organizing the data. It is also used for processing, retrieving, and storing data. There are different basic and advanced types of data structures that are used in almost every program or software system that has been developed. So we must have good knowledge about data structures.

Linear Data Structure:

Data structure in which data elements are arranged sequentially or linearly, where each element is attached to its previous and next adjacent elements, is called a linear data structure. Example: Array, Stack, Queue, Linked List, etc.

    Static Data Structure:

    Static data structure has a fixed memory size. It is easier to access the elements in a static data structure. Example: array.

    Dynamic Data Structure:

    In dynamic data structure, the size is not fixed. It can be randomly updated during the runtime which may be considered efficient concerning the memory (space) complexity of the code. Example: Queue, Stack, etc.

Non-Linear Data Structure:

Data structures where data elements are not placed sequentially or linearly are called non-linear data structures. In a non-linear data structure, we canâ€™t traverse all the elements in a single run only. Examples: Trees and Graphs.

Key Data Structures in C

1. Arrays

Fixed-size sequential collection of elements of the same data type.

Common Operations:

  • Accessing an element: O(1)
  • Insertion/Deletion: O(n) (if shifting is required)
  • Searching: O(n) (linear search) or O(log n) (binary search if sorted).
int arr[5] = {1, 2, 3, 4, 5};
for (int i = 0; i < 5; i++) {
    printf("%d ", arr[i]);
}

2. Linked List

A collection of nodes where each node contains data and a pointer to the next node.

Types:

  • Singly Linked List
  • Doubly Linked List
  • Circular Linked List

Common Operations:

  • Insertion: O(1) at the beginning, O(n) for other positions.
  • Deletion: O(1) at the beginning, O(n) for other positions.
  • Searching: O(n).
struct Node {
    int data;
    struct Node* next;
};

void printList(struct Node* n) {
    while (n != NULL) {
        printf("%d -> ", n->data);
        n = n->next;
    }
    printf("NULL\n");
}

3. Stack

A LIFO (Last In, First Out) data structure.

Common Operations:

  • push(): Add an element to the top.
  • pop(): Remove the top element.
  • peek(): View the top element without removing it.
int stack[MAX];
int top = -1;

void push(int value) {
    if (top == MAX - 1) {
        printf("Stack Overflow\n");
    } else {
        stack[++top] = value;
    }
}

void pop() {
    if (top == -1) {
        printf("Stack Underflow\n");
    } else {
        printf("Popped: %d\n", stack[top--]);
    }
}

4. Queue

A FIFO (First In, First Out) data structure.

Common Operations:

  • enqueue(): Add an element to the rear.
  • dequeue(): Remove an element from the front.
int queue[MAX];
int front = -1, rear = -1;

void enqueue(int value) {
    if (rear == MAX - 1) {
        printf("Queue Overflow\n");
    } else {
        if (front == -1) front = 0;
        queue[++rear] = value;
    }
}

5. Trees

A hierarchical data structure with nodes.

Key Concepts:

  • Binary Trees
  • Binary Search Trees (BSTs)
  • Tree Traversals (Inorder, Preorder, Postorder)
struct Node {
    int data;
    struct Node* left;
    struct Node* right;
};

void inorderTraversal(struct Node* root) {
    if (root != NULL) {
        inorderTraversal(root->left);
        printf("%d ", root->data);
        inorderTraversal(root->right);
    }
}

6. Graphs

A collection of vertices and edges.

Representation:

  • Adjacency Matrix
  • Adjacency List
struct Node {
    int vertex;
    struct Node* next;
};

struct Graph {
    int numVertices;
    struct Node** adjLists;
};
